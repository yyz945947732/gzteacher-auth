### Validator

如果你希望自定义一些权限验证逻辑，你可以使用 `validator`。他会在验证权限编码时串行执行所有验证器，并在权限编码首次匹配到 `match`（正则或字符串） 时，执行对应的 `validate` 来判断权限。

#### 全局验证器 VS 局部验证器

验证器分为全局验证器和局部验证器。

全局验证器通过在 `Provider` 组件中配置 `validator` 属性使用。配置内容将在全局生效。

以下代码将关闭所有权限。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Provider
      auth={{
        global: true,
      }}
      validator={[
        {
          match: /(.*)/,
          validate: (authCode, auth) => false,
        },
      ]}
    >
      <Auth.Wrapper authCode="global">全局生效</Auth.Wrapper>
    </Auth.Provider>
  );
}
```

局部验证器通过使用 `Validator` 组件包裹内容并配置其 `validator` 属性使用。局部验证器将在上下文内替换全局验证器的配置。

以下代码将关闭当前模块的所有权限。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Validator
      validator={[
        {
          match: /(.*)/,
          validate: (authCode, auth) => false,
        },
      ]}
    >
      <Auth.Wrapper authCode="page">局部生效</Auth.Wrapper>
    </Auth.Validator>
  );
}
```

#### 使用案例

你可以用验证器来实现许多丰富的功能。

比如，开启拥有某个前缀的所有权限。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Provider
      auth={{
        love: true,
        peace: true,
      }}
      validator={[
        {
          match: /common\.(.*)/,
          validate: (authCode, auth) => true,
        },
      ]}
    >
      <Auth.Wrapper authCode="peace">世界和平</Auth.Wrapper>
    </Auth.Provider>
  );
}
```

或者，自动在开发环境时开启所有权限。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Validator
      validator={[
        {
          match: /(.*)/,
          validate: (authCode, auth) => {
            if (__ENV__ === "DEV") {
              return true;
            } else {
              return Boolean(auth[authCode]);
            }
          },
        },
      ]}
    >
      <Auth.Wrapper authCode="peace">世界和平</Auth.Wrapper>
    </Auth.Validator>
  );
}
```

如果项目的权限编码具有一定规则，你还可以使用局部验证器来实现更灵活的权限代理，无需使用 `Auth.Proxy` 单独配置每个权限，这样更有利于以后的功能扩展。

```tsx
import { Auth } from "@gzteacher/auth";
import PageA from "./PageA";

export function PageB() {
  return (
    <Auth.Validator
      validator={[
        {
          match: /user\.xxx\.(.*)/,
          validate: (authCode, auth) => {
            const arr = authCode.split(".");
            const lastPart = arr[arr.length - 1];
            if (auth[`teacher.xxx.${lastPart}`]) {
              return true;
            }
            return false;
          },
        },
      ]}
    >
      <PageA />
    </Auth.Validator>
  );
}
```

强制开启某个权限。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Provider
      auth={{
        love: true,
        peace: true,
      }}
      validator={[
        {
          match: "test",
          validate: (authCode, auth) => true,
        },
      ]}
    >
      <Auth.Wrapper authCode="peace">世界和平</Auth.Wrapper>
    </Auth.Provider>
  );
}
```

拥有某个权限时视为拥有所有权限。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Provider
      auth={{
        love: true,
        peace: true,
      }}
      validator={[
        {
          match: /(.*)/,
          validate: (authCode, auth) => {
            if (auth["admin"]) {
              return true;
            }
            return Boolean(auth[authCode]);
          },
        },
      ]}
    >
      <Auth.Wrapper authCode="peace">世界和平</Auth.Wrapper>
    </Auth.Provider>
  );
}
```

或者创造你想要的更丰富的功能。

#### 串行执行

权限编码在经过验证器验证时将按照数组顺序依次匹配，并在匹配到后终止后续匹配，因此你应该将需要优先匹配的验证器放在数组前面。

以下代码在验证权限编码 `peace` 时，第二个和第三个验证器都能匹配到，但因为优先匹配到第二个验证器，所以返回验证结果为 `true`。

```tsx
import { Auth } from "@gzteacher/auth";

export function Example() {
  return (
    <Auth.Provider
      auth={{
        love: true,
        peace: true,
      }}
      validator={[
        {
          match: "war",
          validate: (authCode, auth) => false,
        },
        {
          match: "peace",
          validate: (authCode, auth) => true,
        },
        {
          match: /(.*)/,
          validate: (authCode, auth) => false,
        },
      ]}
    >
      <Auth.Wrapper authCode="peace">世界和平</Auth.Wrapper>
    </Auth.Provider>
  );
}
```
